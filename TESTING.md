## 🧪 Ручне тестування

### Кроки для базового тестування

```bash
# 1. Запустити backend
cd backend
pip install -r requirements.txt
python app.py

# 2. У новому терміналі запустити frontend
cd frontend
python -m http.server 8000

# 3. Відкрити http://localhost:8000
```

### Тестові сценарії

#### Сценарій 1: Додавання замовлення
1. Введіть адресу: "вул. Пушкіна, 10"
2. Виберіть статус: "нове"
3. Натисніть "Додати замовлення"
4. ✅ Замовлення повинно з'явитися у списку

#### Сценарій 2: Зміна статусу
1. Натисніть кнопку "Змінити статус" на замовленні
2. Виберіть новий статус з списку
3. ✅ Статус повинен оновитися

#### Сценарій 3: Видалення замовлення
1. Натисніть кнопку "Видалити" на замовленні
2. Підтвердіть видалення
3. ✅ Замовлення повинно зникнути зі списку

#### Сценарій 4: Офлайн режим
1. Відкрийте додаток в браузері
2. Вимкніть інтернет (F12 → Network → Offline)
3. Додайте нове замовлення
4. ✅ Замовлення повинно зберегтися локально
5. Увімкніть інтернет
6. Натисніть "Синхронізація"
7. ✅ Замовлення повинно синхронізуватися

#### Сценарій 5: PWA встановлення
1. Відкрийте додаток в Chrome/Edge
2. Натисніть на іконку встановлення в адресному барі
3. Виберіть "Встановити"
4. ✅ Додаток повинен встановитися як PWA

## 🔌 Curl команди для API тестування

```bash
# Здоров'я API
curl http://localhost:5000/api/health

# Отримати всі замовлення
curl http://localhost:5000/api/orders

# Додати замовлення
curl -X POST http://localhost:5000/api/orders \
  -H "Content-Type: application/json" \
  -d '{"address":"вул. Шевченка, 5","status":"нове"}'

# Отримати одне замовлення
curl http://localhost:5000/api/orders/1

# Оновити замовлення
curl -X PUT http://localhost:5000/api/orders/1 \
  -H "Content-Type: application/json" \
  -d '{"status":"в дорозі"}'

# Видалити замовлення
curl -X DELETE http://localhost:5000/api/orders/1

# Статистика
curl http://localhost:5000/api/stats
```
