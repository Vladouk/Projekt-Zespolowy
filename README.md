# 📦 Delivery Manager

Prosta aplikacja webowa wspierająca pracę kurierów w zarządzaniu zamówieniami.

---

## 📌 Opis projektu

Delivery Manager to aplikacja internetowa, która umożliwia kurierom:

* przeglądanie listy zamówień
* aktualizację statusu dostawy
* zarządzanie zamówieniami
* organizację pracy

Projekt został stworzony w ramach zespołowego projektu na uczelni.

## 🎯 Cel projektu

Celem projektu jest stworzenie prostego systemu wspomagającego zarządzanie dostawami, który ułatwi pracę kurierów.

---

## ⚙️ Technologie

* Frontend: HTML, CSS, JavaScript
* Backend: Python (Flask)
* Baza danych: SQLite
* Kontrola wersji: Git, GitHub

---

## 🚀 Funkcjonalności

* dodawanie zamówień
* wyświetlanie listy zamówień
* zmiana statusu zamówienia
* usuwanie zamówień
