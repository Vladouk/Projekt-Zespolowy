# Contributing Guide

Дякуємо за інтерес до Delivery Manager! 🎉

## 🤝 Як взяти участь

### 1. Форк та Clone

```bash
# Форк на GitHub
# Потім clone ваш fork
git clone https://github.com/YOUR_USERNAME/Projekt-Zespolowy.git
cd Projekt-Zespolowy

# Додайте upstream
git remote add upstream https://github.com/Vladouk/Projekt-Zespolowy.git
```

### 2. Створіть Feature Branch

```bash
git checkout -b feature/назва-функції
# або
git checkout -b bugfix/назва-баґу
# або
git checkout -b docs/назва-документації
```

### 3. Робіть Зміни

- Слідуйте стилю коду
- Добре коментуйте код
- Тестуйте локально

### 4. Commit та Push

```bash
git add .
git commit -m "🎯 type: Опису вашої зміни"
git push origin feature/назва-функції
```

### 5. Pull Request

- Відкрийте PR на GitHub
- Опишіть що ви робили
- Посилайтеся на Issue якщо є

## 📋 Commit Message Format

```
<emoji> <type>: <subject>

<body>

<footer>
```

### Emoji Опції

- 🚀 feat - нова функція
- 🐛 fix - виправлення баґу
- 📚 docs - документація
- 🎨 style - форматування
- ♻️ refactor - рефакторинг
- 🧪 test - тести
- 🔧 chore - конфігурація
- ⚡ perf - продуктивність

### Приклади

```
🚀 feat: Додано аутентифікацію користувачів

Реалізована базова аутентифікація з JWT токенами.
Користувачі можуть реєструватися та логінитися.

Closes #123
```

```
🐛 fix: Виправлено баґ синхронізації офлайн даних

Проблема: Офлайн дані не синхронізувались при відновленні з'єднання
Рішення: Добавлена перевірка pending changes при reconnect

Closes #456
```

## 🎯 Кодекс поведінки

### Наше Зобов'язання

Ми як учасники, контриб'ютори та лідери зобов'язуємось
дотримуватись етики та поваги до всіх.

### Наші Стандарти

Поведінка яка робить до вітання в нашому проекту:

- Використання гостинної та інклюзивної мови
- Повага до різних точок зору та досвіду
- Прийняття конструктивної критики
- Фокус на те, що найкраще для спільноти
- Проявлення емпатії до учасників

### Забіг Поведінка

Приклади неприйнятної поведінки включають:

- Використання сексуальної мови або образів
- Тролінг, образливі/дискримінуючі коментарі
- Личні або політичні атаки
- Харасмент публічно чи приватно
- Публікація приватної інформації без дозволу
