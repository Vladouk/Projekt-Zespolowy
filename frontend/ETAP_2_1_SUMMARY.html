Метод 1: На дійсному сайті

  1. Відкрийте http://localhost:8000/login.html
  2. Введіть учетні дані: admin / admin123
  3. Нажміть "Войти"
  4. На дашборді нажміть кнопки:
    • 📊 Dashboard
    • 📦 Orders
    • ⚙️ Settings
  5. 🚪 Logout повинен повернути на login.html

Метод 2: Тестова сторінка

Відкрийте http://localhost:8000/test-buttons.html

Тут ви можете тестувати:

  • Direct onclick
  • Event Listener
  • Navigation Links
  • localStorage перевірка
